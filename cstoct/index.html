How to use the converter?

  • Export time to a file from CSTimer
  • Open this file in the converter
  • The file is immediately converted
  • Select a session content (ctrl+a)
  • Copy the content into the clipboard (ctrl+c)
  • In the Cubing Time timer
    • Click on "Settings"
    • Click on "Import solves"
    • Click on "By text"
    • Paste (ctrl+v) the session content
    • Click on "OK"
